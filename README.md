<H3>Installation:</H3>
1) Install python3 and mysql-server on Linux
2) Execute: cat create_db.sql | sudo mysql
3) Copy decimalfixpoints in /var/lib/mysql-files
4) execute cat import_csv.sql | mysql -uadmin
